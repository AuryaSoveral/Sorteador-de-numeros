## Sorteador Dev

Projeto de sorteador de números aleatórios desenvolvido com HTML, CSS e JavaScript.

O usuário informa um número inicial e um número final, e o sistema realiza um sorteio dentro desse intervalo.

## Tecnologias 

- HTML5
- CSS3
- JavaScript

## Funcionalidades

- Sorteio de números aleatórios
- Validação dos campos
- Interface simples e responsiva

## Aprendizados

Nesse projeto foram praticados conceitos como:

- Manipulação do DOM
- Eventos no JavaScript
- Math.random()
- Validação de inputs
- Estilização com CSS
- Responsividade básica
